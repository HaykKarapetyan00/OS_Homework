Напишите 4 программы и Makefile для их сборки

Программа init
Создает разделяемую память размером для хранения массива размера 300 и заполняет нулями, используя ранее созданный файл для генерации ключа. Если по ключу память уже существует или не может быть создана, нужно вывести причину ошибки и завершить процесс с кодом 1. Обратите внимание на выбор флагов для shmget.

Программа destroy
Отмечает разделяемый сегмент как сегмент для удаления. Для этого можно использовать функцию shmctl с параметром cmd равным IPC_RMID. Затем удаляет временный регулярный файл, созданный программой init.

Программа check
На вход получает число N подключает в свое адресное пространство разделяемый сегмент и используя решето проверяет является ли число и простым. В зависимости от ответа печатает "Yes" или "No". В случае ошибок подключения разделяемой памяти вывести причину и завершить процесс с кодом 1.

Программа set
Программа подключает в свое адресное пространство разделяемый сегмент и строит решето Эратосфена
